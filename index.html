Github

O Github é uma espécie de "rede social" de programadores. Com esta plataforma podemos subir os nossos projetos e deixar outras pessoas colaborarem.

Para fazer upload no repositório no GitHub primeiro precisamos fazer um vínculo do git no nosso projeto com o GitHub

Comando para linkar o git com o GitHub

git remote add origin git@github.com:RNSDaniel/aula4.git - "link do repositório no GitHub"

git push -u origin master

Este comando é para subir as alterações feitas no git para o github, atentar que as vezes o "master" pode ser "main" dependendo da versao, assim substituir no código quando necessário.

Para atualizar o repositório local com as alterações salvas no repositório do github utilizamos o git pull.

Podemos também pegar um repositório que estaja disponível no github e baixar para usar no nosso computador, podendo ser um repositório seu ou de outro usuário.

Para isso, no terminal, utilizamos o comandoi git clone seguido da URL do repositório.

git clone https://github.com/minhacontagmail/primeiroprojeto.git

GitHub Pages

O GitHub lhe permite publicar seus projetos on-line. Para gerar uma Github page, você deve:

  1. Acesse seu repositório no GitHub e vá em "Settings"
  2. Clique em "Pages"
  3. Selecione a branch que será publicada (master ou main)
  4. Salve as alterações. O GitHub irá publicar sua página e disponibilizar o link de acesso.
